\# Hospital Management System



A full-stack Hospital Management System built with Spring Boot and React JS.



\## 🖥️ Screenshot

!\[Hospital Management System](screenshot.png)



\## 🛠️ Tech Stack

\- \*\*Frontend:\*\* React JS

\- \*\*Backend:\*\* Java Spring Boot

\- \*\*Database:\*\* MySQL



\## 🚀 How to Run



\### Backend

```bash

cd backend

mvn spring-boot:run

```



\### Frontend

```bash

cd frontend

npm install

npm start

```



\## 📌 Features

\- Patient Registration

\- Doctor Management

\- Appointment Booking

\- User Login \& Authentication
